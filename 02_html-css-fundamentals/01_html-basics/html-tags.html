Headings

Headings from `h1` to `h6` identify blocks and sections.
Every page must have one `h1`

Paragraphs

Paragrapsh of text go in `p` tags

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in the `ul` tag.
Each list item go in the `li` tag.

Types of lists:

  • Unordered lists `(ul)`
  • Ordered lists `(ol)`

Ordered Lists

Ordered lists are numbered and go in the `ol` tag.
Each list item go in the `li` tag.

Anchors (i.e. links)

Anchor tags create links to other pages.
The URL goes in the `href` attribute.

Go to DuckDuckGo to search for more information

Setting `target="_blank"` will open the link in a new window.
See MDN for more details

Anchor tags can also create links to a new location on the same page.
Instead of putting the URL in `href` attribute, just use the `id` of the element you want to navigate to.
This allows you to jump to the top of the page.

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags.
This shows up as italic text in most browsers.

Bold

You can also use the `strong` element to emphasize text.
This shows up as bold text in most browsers.
